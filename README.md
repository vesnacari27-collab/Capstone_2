## Emporium Bookstore Sales Analysis

This capstone project analyzes sales performance across two sales territories for Emporium
Bookstore over a 48-month period (January 2022 – December 2025). EmporiUm is a student bookstore chain that sells tech products, art supplies, and other products in-store and online. I have been assigned 2 territories to work on which are Territory 1 is South Carolina and Territory 2 is New York. 

The goal isto identify revenue trends and produce actionable marketing recommendations using Python-based data analysis. I first made sure to save all the files I was given and I used the project checklist to help guide me.
